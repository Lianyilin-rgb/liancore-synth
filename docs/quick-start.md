# LianCore V3 快速入门指南

> 5 分钟制作你的第一个音色 | 版本 3.0.0

---

## 欢迎！

本指南将带你快速上手 LianCore V3，在 5 分钟内完成从安装到制作第一个音色的全过程。

---

## 第一步：安装和加载（1 分钟）

### 安装

1. 运行安装包 `LianCoreV3_Setup.exe`（Windows）或 `LianCoreV3.pkg`（macOS）
2. 按向导完成安装

### 加载插件

1. 打开你的 DAW（Ableton Live、FL Studio、Cubase、Studio One 等）
2. 创建一条乐器轨道（Instrument Track）
3. 在插件列表中找到 **LianCore V3**（按分类查找：合成器 / Synth）
4. 双击加载，插件界面将自动显示

> **提示**：如果找不到插件，请在 DAW 中执行"重新扫描插件"操作。

---

## 第二步：加载一个出厂预设（30 秒）

1. 点击界面顶部中央的预设名称区域
2. 在预设浏览器中，选择一个分类：
   - **风格**：Bass / Lead / Pad / Keys 等
   - **情绪**：Bright / Dark / Warm 等
3. 双击一个预设名称（例如 "Warm Pad 42"）
4. 弹奏 MIDI 键盘或 DAW 的虚拟键盘，你应该能听到声音了！

> **提示**：500 个精选出厂预设覆盖 10 种风格 × 8 种乐器 × 10 种情绪，随便挑一个开始创作！

---

## 第三步：调整基本参数（1 分钟）

### 试试这几个旋钮，感受音色变化：

1. **滤波器截止频率（Filter Cutoff）**
   - 顺时针：音色更明亮（高频通过）
   - 逆时针：音色更暗沉（高频被削减）
   - 试试：加载一个 Pad 预设，缓慢转动 Cutoff 旋钮

2. **包络起音（Attack）**
   - 增大：声音渐强（像弦乐慢慢拉响）
   - 减小：声音立即触发（像钢琴敲击）
   - 试试：把 Attack 调到 500ms，感受 Pad 音色的渐变

3. **包络释放（Release）**
   - 增大：松开琴键后声音慢慢消失
   - 减小：松开琴键后声音立即停止
   - 试试：把 Release 调到 2 秒，弹奏然后松开

4. **混响（Reverb）**
   - 在效果器区域找到 Reverb 旋钮
   - 调大：声音更有空间感，像在大厅里演奏
   - 试试：加载一个 Pluck 音色，加入 50% 混响

---

## 第四步：切换合成引擎（1 分钟）

在振荡器区域，试试切换合成类型：

1. **默认是减法合成（Subtractive）** — 经典模拟合成器音色
2. 点击下拉菜单，切换到 **FM 合成** — 听听金属感/钟声类音色
3. 切换到 **波表合成（Wavetable）** — 听听动态扫描的音色
4. 切换到 **粒子合成（Granular）** — 试试创造纹理音色

每种合成类型都有独特的音色特征，多试试！

---

## 第五步：保存你的音色（30 秒）

1. 调整参数到满意后，点击 **保存** 按钮（或 Ctrl+S）
2. 填写：
   - **名称**：给你的音色起个名字
   - **风格**：选择风格分类（Bass/Lead/Pad 等）
   - **标签**：添加关键词，如 "warm, analog, retro"
3. 点击 **保存**

你的预设已保存到本地，随时可以加载使用！

## 进阶探索（1 分钟）

完成以上 5 步后，你已经掌握了 LianCore V3 的基础操作。接下来可以尝试：

| 功能 | 怎么玩 | 位置 |

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存预设 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| 双击参数 | 恢复默认值 |
| 右键参数 | MIDI 学习 / 自动化设置 |

## 遇到问题？

- **没声音**：检查 DAW 轨道监听是否开启，MIDI 通道是否正确
- **CPU 过高**：降低复音数（菜单栏 → 设置 → 引擎），关闭 4x 过采样
- **插件未识别**：重新扫描插件，检查安装路径
