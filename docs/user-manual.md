## 1. 概述

### 1.1 关于 LianCore V3

LianCore V3 是一款专业级软音源合成器，采用先进的混合合成引擎，支持 VST3、AU 和 AAX 格式。所有功能完全端侧运行，无需网络连接，确保您的创作数据始终安全。

### 1.2 核心特性

| 特性 | 说明 |
|------|------|
| 6 种合成引擎 | 减法、FM、波表、粒子、加法、物理建模 |
| 32 复音 | 每个音符独立调制，支持 MPE |
| 12 种效果器 | 混响、延迟、合唱、失真、压缩、均衡等 |
| AI 音色分析 | 从音频中提取音色特征，生成匹配参数 |
| AI 波形生成 | 自动生成和优化波表 |
| 100 万+ 预设库 | 500 个精选出厂预设，覆盖 10 风格 × 8 乐器 × 10 情绪 |
| 混沌调制 | 7 种混沌系统，创造复杂 LFO 调制 |
| 粒度合成 | 10 种包络类型，AI 密度控制 |
| MPE 录音/回放 | 标准 MIDI 文件，保留 MPE 元数据 |
| 微音调谐 | .tun 和 .scl 文件支持 |
| 4x 过采样 | 抗混叠，专业音质 |

## 2. 安装

### 2.1 Windows 安装

1. 下载安装包 `LianCoreV3_Setup.exe`
2. 双击运行安装程序
3. 选择安装路径（默认：`C:\Program Files\LianCore\`）
4. 选择需要安装的插件格式：
   - VST3（推荐）
   - AAX（如有 Pro Tools）
5. 点击"安装"，等待完成
6. 启动 DAW，重新扫描插件

**VST3 安装路径**：`C:\Program Files\Common Files\VST3\LianCore.vst3`

### 2.2 macOS 安装

1. 下载 `LianCoreV3.pkg` 安装包
2. 双击运行，按向导完成安装
3. 插件自动安装到标准路径：
   - VST3：`~/Library/Audio/Plug-Ins/VST3/LianCore.vst3`
   - AU：`~/Library/Audio/Plug-Ins/Components/LianCore.component`

### 2.3 手动安装（高级用户）

如需手动安装，将对应格式的插件文件复制到 DAW 的插件目录，然后重新扫描插件。

### 2.4 卸载

- **Windows**：控制面板 → 程序和功能 → LianCore V3 → 卸载
- **macOS**：删除对应插件目录下的 LianCore 文件

## 3. 激活

### 3.1 激活方式

LianCore V3 使用本地密钥文件激活，无需联网。

1. 首次加载插件时，将显示激活窗口
2. 点击"浏览"选择您的许可证文件（`.lic` 格式）
3. 点击"激活"完成验证
4. 激活成功后，许可证信息存储在本地

### 3.2 许可证管理

- 许可证文件与机器绑定，不可跨设备使用
- 如需更换设备，请联系客服获取新的许可证文件
- 激活信息存储在 `%APPDATA%\LianCore\license.dat`（Windows）或 `~/Library/Application Support/LianCore/license.dat`（macOS）

## 4. 基本操作

### 4.1 加载插件

1. 在 DAW 中创建乐器轨道
2. 在插件列表中找到 "LianCore V3"（分类：合成器 / Synth）
3. 加载插件，GUI 将自动显示

### 4.2 界面布局

```
┌──────────────────────────────────────────────┐
│  菜单栏：预设 / 视图 / 设置 / 帮助             │
├──────────────────────────────────────────────┤
│                   波形显示区                    │
├──────────────┬──────────────┬────────────────┤
│   振荡器区域  │   滤波器区域   │   放大器区域    │
├──────────────┼──────────────┼────────────────┤
│   调制矩阵    │   LFO 区域    │   效果器区域    │
├──────────────┴──────────────┴────────────────┤
│              键盘 / MIDI 监控                  │
└──────────────────────────────────────────────┘
```

### 4.3 参数控制

- **旋钮（Knob）**：鼠标上下拖动调整数值，按住 Shift 微调，按住 Ctrl 恢复默认
- **推子（Slider）**：鼠标拖动，支持 Shift 微调
- **按钮（Button）**：单击切换开关状态
- **下拉菜单（Combo Box）**：单击展开选项
- **右键菜单**：右键点击任何参数可进行 MIDI 学习、自动化设置

### 4.4 MIDI 学习

1. 右键点击目标参数
2. 选择"MIDI 学习"
3. 转动 MIDI 控制器上的旋钮/推子
4. 学习完成后，参数自动绑定

### 4.5 自动化

所有参数均支持 DAW 自动化。在 DAW 中启用自动化录制后，直接调整参数即可自动创建自动化轨道。

## 5. AI 功能使用

### 5.1 AI 音色分析

**功能**：从音频文件中分析音色特征，自动生成合成参数来匹配目标音色。

**使用步骤**：
1. 点击菜单栏 → AI → 音色分析
2. 点击"导入音频"，选择 WAV 或 MP3 文件
3. 点击"分析音色"，等待处理完成
4. 在右侧面板查看分析结果：
   - 频谱特征（谐波分布、共振峰）
   - 时域特征（包络、瞬态响应）
   - 推荐合成参数
5. 点击"应用参数"将分析结果应用到当前音色

**支持格式**：WAV（16/24/32 bit）、MP3、FLAC、AIFF

**本地处理**：所有分析在本地完成，音频文件不会上传到任何服务器。

### 5.2 AI 波形生成

**功能**：使用 AI 算法自动生成或优化波表。

**使用步骤**：
1. 点击菜单栏 → AI → 波形生成
2. 选择生成模式：
   - **从头生成**：输入参数（波形数量、复杂度、谐波丰富度）
   - **从音频学习**：导入音频文件，提取波形特征
   - **混合生成**：融合两个现有波形
3. 点击"生成"，等待计算完成
4. 在波形预览中查看结果
5. 点击"保存到波表"将生成的波形添加到当前波表组

### 5.3 AI 预设推荐

**功能**：根据当前演奏的音符或导入的 MIDI，自动推荐合适的预设。

**使用步骤**：
1. 点击菜单栏 → AI → 预设推荐
2. 选择推荐模式：
   - **基于风格**：选择目标音乐风格
   - **基于音色**：上传参考音频
3. 系统将显示 5-10 个最匹配的预设
4. 点击预设名称即可加载

## 6. 预设管理

### 6.1 浏览预设

1. 点击界面顶部预设名称区域
2. 预设浏览器将显示所有预设，按分类排列
3. 分类筛选：
   - **风格**：Bass / Lead / Pad / Pluck / Keys / FX / Synth / Arp / Chord / Drums
   - **乐器**：Analog / FM / Wavetable / Granular / Additive / Subtractive / Physical / Hybrid
   - **情绪**：Bright / Dark / Warm / Cold / Aggressive / Soft / Evolving / Static / Ethereal / Punchy
4. 双击预设名称加载

### 6.2 保存预设

1. 调整音色参数至满意
2. 点击"保存"按钮（或 Ctrl+S）
3. 填写预设信息：
   - 名称
   - 分类（风格/乐器/情绪）
   - 标签（逗号分隔）
   - 描述
   - 作者
4. 点击"保存"

### 6.3 导入/导出预设

**导出**：
1. 菜单栏 → 预设 → 导出
2. 选择导出格式：单个预设（.lcpreset）或批量导出（文件夹）
3. 选择保存路径

**导入**：
1. 菜单栏 → 预设 → 导入
2. 选择 .lcpreset 文件或包含预设的文件夹
3. 点击"导入"

### 6.4 收藏与评分

- 点击预设旁的星形图标收藏
- 评分 1-5 星，用于筛选高质量预设
- 收藏和评分自动保存

## 7. 合成引擎

### 7.1 振荡器（Oscillator）

每个声音最多支持 3 个振荡器，每个振荡器可选择 6 种合成类型：

| 类型 | 说明 | 典型用途 |
|------|------|----------|
| 减法合成 | 经典模拟合成，通过滤波器削减谐波 | Bass、Lead、Pad |
| FM 合成 | 频率调制，产生金属感/钟声类音色 | Keys、Bell、FX |
| 波表合成 | 扫描波表，创建动态音色 | Pad、Evolving |
| 粒子合成 | 将音频拆分为微小粒子重组 | Texture、Ambient |
| 加法合成 | 叠加多个正弦波谐波 | Organ、纯净音色 |
| 物理建模 | 模拟真实乐器物理特性 | String、Wind |

**Unison 齐奏**：1-8 个声部，可调节失谐量和立体声宽度。

### 7.2 滤波器（Filter）

| 类型 | 斜率 | 说明 |
|------|------|------|
| 低通 (LPF) | 12/24 dB/oct | 削减高频，保留低频 |
| 高通 (HPF) | 12/24 dB/oct | 削减低频，保留高频 |
| 带通 (BPF) | 12/24 dB/oct | 保留中心频率附近 |
| 陷波 (Notch) | 12 dB/oct | 削减中心频率 |
| 峰值 (Peak) | - | 提升中心频率 |
| 低架 (Low Shelf) | - | 提升/削减低频 |
| 高架 (High Shelf) | - | 提升/削减高频 |

**滤波器包络**：ADSR 包络可调制截止频率，创造动态音色变化。

### 7.3 放大器（Amplifier）

- ADSR 包络控制音量变化
- 力度（Velocity）控制音量响应
- 键盘跟踪（Key Tracking）控制不同音高的音量差异

## 9. MPE 支持

### 9.1 什么是 MPE

MPE（MIDI Polyphonic Expression）允许每个音符独立控制弯音、压力、释放速度等参数，创造更富有表现力的演奏。

### 9.2 启用 MPE

1. 菜单栏 → 设置 → MIDI
2. 勾选"启用 MPE"
3. 设置 MPE 区域（默认：Zone 1, 15 通道）
4. 在 DAW 中配置 MPE 输入设备

### 9.3 MPE 录音/回放

- **录音**：点击"录音"按钮，演奏将保存为标准 MIDI 文件（保留 MPE 元数据）
- **回放**：点击"加载 MIDI"，选择录音文件播放
- **导出**：录音可导出为 .mid 文件，在其他 DAW 中使用

- 阿拉伯音阶 (Maqam)
- 印度音阶 (Raga)

## 12. 粒度合成

### 12.1 启用粒度合成

在振荡器类型中选择"粒子合成 (Granular)"。

### 12.2 参数

| 参数 | 说明 | 范围 |
|------|------|------|
| 粒子密度 | 每秒生成的粒子数量 | 1-200 |
| 粒子长度 | 每个粒子的持续时间 | 5-500 ms |
| 位置 | 在采样中的读取位置 | 0-100% |
| 位置随机 | 播放位置的随机偏移 | 0-100% |
| 音高随机 | 每个粒子的音高随机偏移 | ±24 半音 |
| 包络类型 | 10 种：Gaussian/Hann/Blackman/Exponential/Triangle/Trapezoid/Welch/Cosine/Flat/Parabolic |
| 方向 | 正向/反向/双向/随机 |

### 12.3 AI 密度控制

- 点击"AI 密度"，系统自动分析输入音频并推荐最佳密度值
- 适用于创造自然、有机的纹理音色

## 13. 混沌调制

### 13.1 混沌 LFO 类型

| 类型 | 说明 | 典型用途 |
|------|------|----------|
| Logistic Map | 经典混沌系统 | 不可预测的调制 |
| Lorenz | 三变量混沌系统 | 复杂多维度调制 |
| Rossler | 螺旋混沌系统 | 缓慢变化的调制 |
| Henon | 二维混沌映射 | 离散跳跃调制 |
| Tent Map | 分段线性混沌 | 尖锐的调制变化 |
| Circle Map | 环形混沌映射 | 循环调制 |
| Duffing | 非线性振荡器 | 谐波丰富的调制 |

### 13.2 参数

- **混沌度（Chaos）**：控制混沌程度，0=周期，1=完全混沌
- **速率（Rate）**：调制速度
- **深度（Depth）**：调制幅度
- **平滑（Smooth）**：输出平滑处理

## 14. 故障排除

### 14.1 常见问题

| 问题 | 解决方案 |
|------|----------|
| 插件未被 DAW 识别 | 重新扫描插件；检查安装路径是否正确 |
| 无声音输出 | 检查 MIDI 通道；确认 DAW 已激活轨道监听 |
| 高 CPU 占用 | 降低复音数；关闭 4x 过采样；减少效果器数量 |
| 预设加载失败 | 检查预设数据库是否完整；重新安装预设库 |
| 激活失败 | 确认许可证文件路径正确；检查文件是否损坏 |
| GUI 显示异常 | 更新显卡驱动；尝试关闭 DAW 的 GPU 加速 |

### 14.2 日志文件

- **Windows**：`%APPDATA%\LianCore\logs\`
- **macOS**：`~/Library/Logs/LianCore/`

### 14.3 技术支持

- 提交 Issue：https://github.com/Lianyilin-rgb/liancore-synth/issues
- 日志文件有助于快速定位问题，提交时请附上日志
