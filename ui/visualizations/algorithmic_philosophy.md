# LianCore 算法艺术哲学

## 核心理念：声音的视觉化映射

声音是时间与频率的互动。LianCore 的算法艺术将合成器信号链中的三个核心维度——波形、频谱、调制——转化为可交互的视觉体验。

### 1. 波表可视化（Wavetable）
- **哲学**：波表是声音的"指纹"，每个波形周期承载着独特的谐波结构
- **算法**：基于傅里叶级数合成，叠加奇次/偶次谐波，支持波形变形（warping）和相位偏移
- **视觉语言**：使用渐变色彩编码振幅，网格线标识零交叉点，拖拽交互支持实时波形塑造

### 2. 频谱图（Spectrogram）
- **哲学**：频谱是声音的"解剖图"，展示频率能量随时间的演化
- **算法**：基于 FFT（快速傅里叶变换），将时域信号转换到频域，使用对数频率轴
- **视觉语言**：瀑布图（waterfall）展示时间维度，色彩映射使用冷暖渐变（低频暖色→高频冷色）

### 3. LFO 动态曲线（LFO Curves）
- **哲学**：LFO 是声音的"呼吸"，低频振荡器赋予音色运动感和生命力
- **算法**：多种波形生成（正弦/三角/锯齿/方波/随机/S&H），支持频率调制和相位偏移
- **视觉语言**：实时轨迹追踪，历史残影（trail），当前相位点高亮，多 LFO 叠加显示

### 种子随机性
- 所有随机参数使用确定性种子（p5.js `randomSeed()`），保证审美一致性
- 支持用户在"秩序"（规则波形）与"混沌"（噪声调制）之间探索

### 交互设计
- 模式切换：波表 / 频谱 / LFO
- 参数调节：频率、谐波级数、波形类型、调制深度
- 鼠标交互：拖拽波形节点、缩放频谱范围、暂停/播放动画
